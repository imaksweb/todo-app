# Todo App with API

This application allows you to create and store your own todos list.

Functionality:
  - create new todo
  - delete todo
  - edit todo
  - mark todo as completed
  - search todo
  - filter todos by `completed`, `active` or `all`

Technologies:
  - React
  - Typescript
  - Bulma
  - REST API
  - SASS(SCSS)

Working example you can find here
